## Overview

The WoW Auction Tracker is a comprehensive application designed to monitor, analyze, and track World of Warcraft auction house data. It provides real-time price tracking, historical analysis, and market trend insights to help players make informed trading decisions.

### Key Features

- **Real-time Auction Monitoring**: Track live auction house data
- **Price History Analysis**: Historical price tracking and trend analysis
- **Market Alerts**: Customizable alerts for price changes and opportunities
- **Data Export**: Export data in various formats (CSV, JSON, Excel)
- **Multi-realm Support**: Track auctions across multiple WoW realms
- **Item Database**: Comprehensive item information and statistics

### Architecture

The application follows a modular architecture with clear separation of concerns:

- **Frontend**: Modern web interface built with React/TypeScript
- **Backend**: RESTful API built with Node.js/Express
- **Database**: PostgreSQL for data persistence
- **Data Collection**: Automated auction house data scraping
- **Analytics Engine**: Real-time data processing and analysis

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables
4. Start the development server: `npm run dev`
5. Access the application at `http://localhost:3000`

For detailed setup instructions, see [Getting Started](./getting-started.md).
